# Zythor Privacy Policy

Last Updated: June 2026

Zythor is a Discord bot that stores limited data necessary to provide its features and services.

## Data Collected

* Discord User IDs
* Discord Server IDs
* Channel IDs
* Role IDs
* Server configuration settings
* Moderation records
* Reaction role configurations
* Leveling and profile data

## How Data Is Used

The collected data is used solely to provide bot functionality, including moderation, server management, reaction roles, leveling systems, welcome messages, and other features requested by server administrators.

## Data Storage

Data is stored securely in MongoDB and is only accessible by authorized bot administrators.

## Data Sharing

Zythor does not sell, rent, or share user data with third parties except where required by law.

## Data Deletion

Users may request deletion of their stored data by contacting the bot developer through the support server.

## Contact

For privacy-related requests, contact the Zythor support team through the official support server.

By using Zythor, users agree to this Privacy Policy.
